# Getting Started

To run this code locally:

```
git clone https://github.com/greatsamist/Tuks-Dapp
```

```
cd Tuks-Dapp
```

### For backend

```
cd  Tuks-Dapp/backend
```

```
yarn install
```

### For frontend

```
cd  Tuks-Dapp/frontend/clubry
```

```
yarn install
```

```
yarn run dev
```
